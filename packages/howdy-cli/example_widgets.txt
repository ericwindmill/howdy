## Widgets

### Select

* Title, Description and Options can be styled separately
* Hovered style can be passed in

```template
Title
Description 
  ❯ Option1  
    Option2
    Option3
```

### Select.multi

* Title, Description and Options can be styled separately
* Hovered style can be passed in

```template
Title
Description 
  ❯ ◯ Option1  
    ◉ Option2
    ◯ Option3
```

## Form

* Form displays indented by 3, and the focused element has ' | ' down 
  the side. The element not focused doesnt

```example
 | Title               
 | HelperText                    
 | ❯ 

   Title
   Description 
    ❯ ◯ Option1  
      ◉ Option2
      ◯ Option3
```

* Form is a "screen" with widgets that change, but are always there

```example

  Title or Description.dim (optional, one line)

  <-----
  
  
    WIDGETS
  
  
  
  ---->

  error or warning line (can be blank, warning is yellow, error is red)
  guide.dim (e.g. "up/down to select, enter for next" changes depend on focused element)    1/2 (page number, aligned right)
```
